# Introdução
 
Bem vindo a Nextra! Este é um modelo básico de documentos. Você pode usá-lo como ponto de partida para seu próprio projeto :)

## O que é Nextra?

Uma estrutura de geração de sites **simples**, **poderosa** e **flexível** com tudo o que você adora do Next.js.

## Documentação

A documentação está disponível em [https://nextra.site](https://nextra.site).
